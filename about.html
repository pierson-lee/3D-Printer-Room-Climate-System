About this project

This 3D Printer climate system monitors the temperature and humidity of my 3D printing room in real time using an ESP32 and humiture sensor.

The collected data is sent to this website where it can be monitored remotely and analyzed for the effect it may have on print quality.

Why I built this

I started noticing failed prints due to warping, stringing, and detaching. The overall cause was poor temperature and humidity control.

Certain filament types react differently to humidity and temperature, especially during long print jobs.

Being able to monitor these conditions in real-time has allowed me to complete prints with significantly higher success rates.

System Features

  • Real-time temperature and humidity monitoring
  • Filament-specific condition analysis
  • Timelines of ambient conditions
  • Remote access through web dashboard
